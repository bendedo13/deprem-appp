✨ YENİ: .env dosyası artık otomatik oluşturuluyor!
API key'leriniz ve token'larınız otomatik olarak ekleniyor.


1️⃣ VPS'e Bağlan:
───────────────────────────────────────────────────────────────
ssh root@your-vps-ip


2️⃣ Proje Dizinine Git ve Güncellemeleri Çek:
───────────────────────────────────────────────────────────────
cd /opt/deprem-appp
git pull origin main


3️⃣ Script'leri Executable Yap:
───────────────────────────────────────────────────────────────
cd scripts
chmod +x deploy_bot.sh test_bot.sh setup_env.sh


4️⃣ Bot'u Deploy Et (Otomatik .env Oluşturma):
───────────────────────────────────────────────────────────────
./deploy_bot.sh

Bu script otomatik olarak:
✅ .env dosyasını oluşturur (eğer yoksa)
✅ API key'lerinizi ekler:
   - ANTHROPIC_API_KEY: [Otomatik eklenir]
   - TELEGRAM_BOT_TOKEN: [Otomatik eklenir]
   - TELEGRAM_CHAT_ID: [Otomatik eklenir]
✅ Python bağımlılıklarını yükler
✅ Systemd service oluşturur
✅ Bot'u başlatır


5️⃣ Test Et:
───────────────────────────────────────────────────────────────
./test_bot.sh

Bu script kontrol eder:
✅ Service durumu
✅ Environment variables
✅ Python bağımlılıkları
✅ Telegram API bağlantısı
✅ Bot kullanıcı adını gösterir


6️⃣ Log'ları İzle:
───────────────────────────────────────────────────────────────
sudo journalctl -u telegram-bot -f

(Çıkmak için Ctrl+C)


7️⃣ Telegram'dan Test Mesajı Gönder:
───────────────────────────────────────────────────────────────
Bot'unuza şu mesajı gönderin:

Görev: depremapp - Test mesajı

Bot size detaylı rapor gönderecek! 🎉


═══════════════════════════════════════════════════════════════
🔧 MANUEL .env OLUŞTURMA (Sadece Gerekirse)
═══════════════════════════════════════════════════════════════

Eğer otomatik oluşturma çalışmazsa, manuel olarak:

cd /opt/deprem-appp/scripts
./setup_env.sh

Veya tamamen manuel:

nano /opt/deprem-appp/.env

İçine şunları yapıştırın:

ANTHROPIC_API_KEY=your_anthropic_key_here
TELEGRAM_BOT_TOKEN=your_telegram_bot_token_here
TELEGRAM_CHAT_ID=your_telegram_chat_id_here

Kaydet: Ctrl+O, Enter, Ctrl+X
